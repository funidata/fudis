# Border mixins

Load Fudis mixins and tokens to your file in your SCSS file with:

```
@use "@funidata/ngx-fudis" as fudis;
```

## Border

Set the border of an element.

```
@include fudis.border($width, $style, $color)

```

| $width                                                 | $style                                                 | $color                                          |
| ------------------------------------------------------ | ------------------------------------------------------ | ----------------------------------------------- |
| string value from spacing tokens map (_$pixel-tokens_) | string value from border tokens map (_$border-styles_) | string value from colors tokens map (_$colors_) |

```
.my-element {
  @include fudis.border('2px', 'solid', 'primary');
}
```

## Outline

Set the outline of an element.

```
@include fudis.outline($width, $style, $color)
```

| $width                                                 | $style                                                 | $color                                          |
| ------------------------------------------------------ | ------------------------------------------------------ | ----------------------------------------------- |
| string value from spacing tokens map (_$pixel-tokens_) | string value from border tokens map (_$border-styles_) | string value from colors tokens map (_$colors_) |

```
.my-element {
  @include fudis.outline('1px', 'dashed', 'gray-middle');
}
```

## Border radius

Set border radius of an element.

```
@include fudis.border-radius($width)
```

| $width                                                 |
| ------------------------------------------------------ |
| string value from spacing tokens map (_$pixel-tokens_) |

```
.my-element {
  @include fudis.border-radius('2px');
}
```
