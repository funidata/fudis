# Border Mixins

Load Fudis mixins and tokens to your SCSS file with:

```
@use "@funidata/ngx-fudis" as fudis;
```

## Border

Set border of an element.

```
@include fudis.border($width, $style, $color)

```

<table>
  <thead>
    <tr>
      <th>$width</th>
      <th>$style</th>
      <th>$color</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>string value from spacing tokens map (**$pixel-tokens**)</td>
      <td>string value from border tokens map (**$border-styles**)</td>
      <td>string value from colors tokens map (**$colors**)</td>
    </tr>
  </tbody>
</table>

```
.my-element {
  @include fudis.border('2px', 'solid', 'primary');
}
```

### Border None

Border none doesn't need any parameters.

```
@include fudis.border-none;

```
