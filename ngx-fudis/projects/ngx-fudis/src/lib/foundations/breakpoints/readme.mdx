# Breakpoint Mixins

[Figma for Breakpoints](https://www.figma.com/file/a6G6ZD6W3tlq0qmUkdvdp1/Fudis-DS-Foundations?type=design\&node-id=822-394\&mode=design\&t=Z3hBCijTMHJNlDzQ-0)

Load Fudis mixins and tokens to your SCSS file with:

```
@use "@funidata/ngx-fudis" as fudis;
```

## Min-width

When you are using `breakpoint('md')`, your styles will be applied from `$breakpoint-md` and up.

```
@include fudis.breakpoint($breakpoint){}
```

```
.my-element {
  @include fudis.breakpoint('md') {
    // Rules defined here apply from breakpoint 'md' and up
  };
}

```

## Max-width

When you are using `breakpoint-max('md')`, your styles will be applied when viewport is smaller than `$breakpoint-md`.

```
@include fudis.breakpoint-max($breakpoint){}
```
