# Tooltip

Tooltip directive uses Angular Material MatTooltip component as a base. Tooltip is included in IconComponent, where tooltip text is added with `tooltip` property. Optional properties include `tooltipPosition` and `tooltipToggle`.
Regular tooltip is triggered with mousehover and on keyboard focus. This content is forced for keyboard and screenreader users, when element receives focus. Consider using `tooltipToggle` property with long and alternative text contents.
TooltipToggle makes tooltip function as clickable pop-over and it will not distract keyboard and screen reader users.

### Accessibility
- Keyboard and screen reader users can select element that triggeres tooltip
- Tooltip content is read to screen readers when tooltip element receives focus
- Extensive and alternative content tooltips use tooltipToggle. Screen reader users are informed about the toggled hidden content when the element receives focus.
