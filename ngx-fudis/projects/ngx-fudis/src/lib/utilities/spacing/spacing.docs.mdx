# Spacing

Fudis Spacing classes allows you to add responsive margin and paddings to components and to any other native UI elements. With Fudis Spacing classes you can add more space to the elements right , left, top or bottom. Fudis spacing classes also include class names to include horizontal and vertical space simultaneously.
When class is used with a varying breakpoint value, note that the chosen breakpoint refers to the smallest screen size when the style has affect.

## Usage

### With Varying Values for Different Breakpoints
```
fudis - $breakpoint - $direction-attribute - $spacing-token
```
Responsive class name formation start with a prefix 'fudis' and follows a pattern with respective **Fudis Foundation tokens:**<br/>

- [Breakpoint tokens](/docs/foundations-breakpoints--documentation): Screen min widths values from `xs` to `xxl`.
- **Direction attributes:** Name identifiers for margin and padding directions.
- [Spacing tokens](/docs/foundations-spacing--documentation): Sizes from `xxs` to `xxl`, including `none`.


### With Single Value for All Breakpoints
```
fudis - $direction-attribute - $spacing-token
```
For spacing values that apply throught out the different screen sizes simply remove the 'breakpoint token'.
