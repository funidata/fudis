### Accessibility

- Focus is trapped inside the dialog
- Focus returns to dialog launcher when it is closed
- Dialog can be closed with keyboard Escape
- If dialog content does not include actionable element (eg.buttons) dialog content can be focused by setting contentFocus to true.
- Sorting icons have aria-labels to describe how they sort the content
- Table has a caption
