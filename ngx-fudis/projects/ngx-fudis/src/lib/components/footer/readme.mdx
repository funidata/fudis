# Footer

Footer is a static component to be used as application's footer in the bottom of the page.

## Usage Guidelines

Footer is designed to be used with [Link Components](/docs/components-link--documentation).
By using directives `fudisFooterContentRight` and/or `fudisFooterContentLeft` it determines which side of the footer links are displayed.
The only fixed element is Funidata logo with a link to Funidata's homepage and translated alternative text for screen readers.
