Fudis Exercise On this page, you'll find two hands-on exercises designed to help you get familiar with the Fudis component library. Pick either exercise and follow the step-by-step instructions. In the first exercise, you'll build a layout using the Fudis grid and basic components. In the second, you'll learn how to contribute to the Fudis component library yourself. Pre-requirement Before starting the exercises:
  1. Make sure that your have both environments, Storybook and Sandbox, running as documented on .
  2. Create a Git branch with a suitable name for the exercise you are doing. For example, if you are doing the first exercise, we advice you to name your branch with identifier "[your-name]-exercise-one". More detailed instructions on how to name your branch and commit messages are documented on page . But for the sake of this exercise you don't need to stick to them.
