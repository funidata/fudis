Contributing to Fudis Design System Fudis Design System is an open source project and we welcome contributions from everyone. This exercise guides you through the process of contributing to Fudis Design System. In this exercise you will be creating a Storybook story where you utilise Fudis Dialog. As a bonus task, you will get familiar with styling elements by using Core style Library Instructions
