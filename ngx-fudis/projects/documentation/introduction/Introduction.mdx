# Introduction

Fudis Design System provides guidelines, design assets and components that work as the foundation for all of Funidata's products. The purpose of this Angular-based system is to support a sustainable and consistent development and accessible customer experience for our services.

## For Product Developers

<br />

### How to Use Fudis in Your Application

To install Fudis in your application, check [How to Start Using Fudis documentation.](/docs/documentation-introduction-how-to-start-using-fudis--documentation).

For each component's documentation, usage guidelines and examples are documented in component's own Storybook page, e.g. [documentation for Button component](/docs/components-button--documentation).

If you need support, do not hesitate to send message to Fudis developers and maintainers. Most convinient way for people working at Funidata is our [#fudis Slack channel](https://sisudev.slack.com/archives/C03G46K6LG1).

## For Fudis Developers

Instructions and guidelines on how to contribute to development of Fudis, start from [Getting Started](/docs/documentation-development-getting-started--documentation) under **Development** section of documentation.
